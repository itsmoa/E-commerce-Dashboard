E-Commerce Sales Dashboard

Data Visualization | Excel & Power BI | 2024

Project Overview

This interactive dashboard provides a comprehensive overview of e-commerce performance across regions, product categories, and months. It showcases key metrics including Total Sales ($247,352), Average Order Value ($2,473.52), and highlights top-performing categories such as Computing and Networking.

Visuals include sales by region, sales by category, monthly sales trends (Jan–Apr 2024), and top 5 products like Smartphone X and Tablet Ultra. Built using Excel and Power BI, it helps identify growth opportunities and regional performance insights through clean and clear visual storytelling.

Tools Used: Microsoft Excel, Power BI, Data Visualization, Dashboard Design
